# Meeting Notes Automation

An automated system that transforms meeting transcripts into formatted email drafts.

## What It Does

This application automatically:
1. Retrieves meeting recordings and transcripts from Attio CRM
2. Formats meeting notes using Claude AI
3. Creates professional email drafts in Gmail with meeting summaries and action items

## Features

- Automated meeting note formatting
- Professional email templates with action items
- Multi-account Gmail draft creation
- Integration with Attio CRM and Claude AI

## Technology Stack

- AWS Lambda for serverless processing
- Attio API for meeting data
- Claude AI for intelligent formatting
- Gmail API for draft creation

## Use Case

Designed for account managers and teams who need to quickly distribute meeting notes to clients and stakeholders. The system handles the entire workflow from transcript to polished email draft.
